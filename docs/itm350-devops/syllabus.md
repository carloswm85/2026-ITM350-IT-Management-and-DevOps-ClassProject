## Course Overview

**ITM 350 - DevOps**

In today’s fast-paced software landscape, DevOps enables efficient building, deploying, and maintaining of reliable applications.

This course provides a **hands-on, practical foundation** focused on:

- Automation
- Collaboration
- Continuous improvement

Key areas:

- Git (version control, branching, code reviews)
- CI/CD pipelines
- Cloud-based deployments

**Outcome:**
Students gain real-world skills to improve software quality, scalability, and operational efficiency.

---

**Recommendation:**

- Work consistently throughout the week
- Avoid last-minute work

---

## Text & Resources

- No required textbook
- Online materials are used extensively

**Requirement:**

- Stable internet connection

---

## Course Objectives

Upon completion, students will be able to:

- Use **Git** effectively (branching, code reviews, collaboration)
- Implement **automated testing** (unit + integration)
- Build and manage **containerized applications**
- Design and implement **CI/CD pipelines**
- Apply **Infrastructure as Code (IaC)**
